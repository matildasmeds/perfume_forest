== About

PerfumeForest is a tiny catalogue of perfumes. You can 'like' perfumes. The application stores likes in the browser's local storage. 

Ruby on Rails 4.2, Thin webserver, Postgresql, Haml, CSS, JavaScript, RSpec, Capybara, Selenium and Chrome driver.
